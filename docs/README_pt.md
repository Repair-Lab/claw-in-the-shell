## 🌊 O que é GhostShell?

**GhostShell é um sistema operacional de IA relacional.** Enquanto projetos como OpenClaw rodam *sobre* um sistema, GhostShell **é** o sistema. Ele transforma um banco de dados PostgreSQL em um organismo vivo, onde drivers de hardware, sistemas de arquivos e modelos de IA ("Ghosts") se comunicam através de tabelas SQL.

Cada pensamento. Cada movimentação de arquivo. Cada impulso de hardware. Tudo — transações de banco de dados compatíveis com ACID. Indestrutível. Seguro. Consistente.

---

## 🔥 Por que GhostShell e não OpenClaw?

| | OpenClaw | GhostShell OS |
|---|---|---|
| **Arquitetura** | Aplicação em um sistema | **É** o sistema |
| **Persistência** | Memória volátil | Transações ACID — cada pensamento é permanente |
| **Hardware** | APIs externas | Hardware-como-Tabela — `UPDATE cpu SET governor='performance'` |
| **Modelos IA** | Modelo único, requer reinício | Hot-Swap Ghosts — troque LLMs sem perder contexto |
| **Segurança** | Nível de aplicação | 3 camadas de imutabilidade: Núcleo → Runtime → Ghost |
| **Autorreparação** | Manual | Pipeline de reparo autônomo com aprovação humana |

---

## 🔒 Três Camadas de Segurança

**O Ghost pode reparar — mas nunca reconstruir.** Cada mudança proposta passa por:

```
Ghost propõe → Humano aprova → SECURITY DEFINER executa → Log de auditoria
```

## 🦾 Funcionalidades

- [x] **Hardware-como-Tabela** — Controle ventiladores, clock da CPU e discos via `SQL UPDATE`
- [x] **17 Aplicativos Desktop** — Ghost Chat, Loja de Software, Gerenciador LLM e mais
- [x] **Hot-Swap Ghosts** — Troque seu LLM em tempo de execução sem perder contexto
- [x] **Proteção de Imutabilidade** — 176 impressões digitais de schema, registro de violações
- [x] **Pipeline de Reparo** — Ghost propõe → Humano aprova → Execução segura
- [x] **Ponte OpenClaw** — Importe habilidades OpenClaw para um ambiente mais seguro
- [x] **Métricas em Tempo Real** — CPU, RAM, GPU, temperatura via WebSocket
- [x] **Segurança em Nível de Linha** — 71 tabelas com políticas RLS em 5 papéis de banco
- [ ] **Codificação Autônoma** *(Em Progresso)* — Ghost escreve suas próprias migrações SQL
- [ ] **Integração de Visão** *(Planejado)* — Análise de vídeo em tempo real
- [ ] **Ghosts Distribuídos** *(Planejado)* — Múltiplas instâncias Ghost entre nós
