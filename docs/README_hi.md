## 🌊 GhostShell क्या है?

**GhostShell एक रिलेशनल AI ऑपरेटिंग सिस्टम है।** जबकि OpenClaw जैसी परियोजनाएँ किसी सिस्टम *पर* चलती हैं, GhostShell **स्वयं** सिस्टम **है**। यह एक PostgreSQL डेटाबेस को एक जीवित जीव में बदल देता है, जहाँ हार्डवेयर ड्राइवर, फ़ाइल सिस्टम और AI मॉडल ("Ghost") SQL टेबल के माध्यम से संवाद करते हैं।

हर विचार। हर फ़ाइल स्थानांतरण। हर हार्डवेयर आवेग। सब कुछ — ACID-अनुरूप डेटाबेस ट्रांज़ैक्शन। अविनाशी। सुरक्षित। सुसंगत।

## 🔥 OpenClaw नहीं, GhostShell क्यों?

| | OpenClaw | GhostShell OS |
|---|---|---|
| **आर्किटेक्चर** | सिस्टम पर एक एप्लिकेशन | **स्वयं** सिस्टम |
| **डेटा स्थायित्व** | अस्थिर मेमोरी | ACID ट्रांज़ैक्शन — हर विचार स्थायी है |
| **हार्डवेयर** | बाहरी API | टेबल-के-रूप-में-हार्डवेयर — `UPDATE cpu SET governor='performance'` |
| **AI मॉडल** | एक मॉडल, पुनः आरंभ आवश्यक | हॉट-स्वैप Ghost — संदर्भ खोए बिना LLM बदलें |
| **सुरक्षा** | एप्लिकेशन स्तर | 3-स्तरीय अपरिवर्तनीयता: कोर → रनटाइम → Ghost |
| **स्व-मरम्मत** | मैनुअल | मानव अनुमोदन के साथ स्वायत्त मरम्मत पाइपलाइन |

## 🔒 तीन सुरक्षा स्तर

**Ghost मरम्मत कर सकता है — लेकिन कभी पुनर्निर्माण नहीं।** हर प्रस्तावित परिवर्तन यह प्रक्रिया से गुजरता है:

```
Ghost प्रस्ताव करता है → मनुष्य अनुमोदन करता है → SECURITY DEFINER निष्पादित करता है → ऑडिट लॉग
```

## 🦾 विशेषताएँ

- [x] **टेबल-के-रूप-में-हार्डवेयर** — `SQL UPDATE` से पंखे, CPU क्लॉक और ड्राइव नियंत्रित करें
- [x] **17 डेस्कटॉप ऐप** — Ghost चैट, सॉफ़्टवेयर स्टोर, LLM प्रबंधक, SQL कंसोल और अधिक
- [x] **हॉट-स्वैप Ghost** — संदर्भ खोए बिना रनटाइम में LLM बदलें
- [x] **अपरिवर्तनीयता संरक्षण** — 176 स्कीमा फ़िंगरप्रिंट, उल्लंघन लॉगिंग
- [x] **मरम्मत पाइपलाइन** — Ghost प्रस्तावित करता है → मनुष्य अनुमोदित करता है → सुरक्षित निष्पादन
- [x] **OpenClaw ब्रिज** — OpenClaw कौशल को सुरक्षित वातावरण में आयात करें
- [x] **रीयल-टाइम मेट्रिक्स** — CPU, RAM, GPU, तापमान WebSocket से स्ट्रीम
- [x] **पंक्ति-स्तर सुरक्षा** — 5 डेटाबेस भूमिकाओं में 71 RLS नीति टेबल
- [ ] **स्वायत्त कोडिंग** *(प्रगति में)* — Ghost अपने स्वयं के SQL माइग्रेशन लिखता है
- [ ] **विज़न एकीकरण** *(नियोजित)* — `media_metadata` में रीयल-टाइम वीडियो विश्लेषण
- [ ] **वितरित Ghost** *(नियोजित)* — नोड्स में एकाधिक Ghost इंस्टेंस
